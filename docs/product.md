# 内容矩阵

## 内容结构：四维矩阵

每个支柱都要做 4 类页面：

- **主题页**（Topic Hub）：该主题的入口页，2000-3000 字
- **子主题页**（Subtopic）：具体场景，1500-2500 字
- **城市页**（City）：每个城市该主题的实操，1500-2000 字
- **工具页**（Tool）：互动工具，提升 SEO + 留存

!!! example "举例（「支付」支柱）"
    - 主题：`/pay-in-china/` — "中国支付完全指南"
    - 子主题：`/pay-in-china/alipay-for-foreigners/` / `wechat-pay-foreigners/` / `cash-in-china/`
    - 城市：`/pay-in-china/beijing/` / `shanghai/` / `chengdu/`
    - 工具：`/tools/alipay-setup-checker/` / `currency-converter/`

## 十大内容支柱

### 支柱 1：签证（占内容 20%）

- **目标人群**：免签国家（无需看）/ 需签证国家
- **子主题**：
    - 中国签证类型全集（L/M/F/Z/X/Q1/Q2/J1/J2/D...）
    - 144/240 小时过境免签适用城市清单
    - APEC 商务旅行卡
    - 团体签证（ADS）
    - **免签国家实时名单（每月更新）**
- **文章模板**：「X 国人去中国需要签证吗？2026 最新政策」
- **变现**：高价值 lead，可接签证代办服务（CPS 5-15%）

### 支柱 2：通讯上网 ⭐（占 15%，高搜索）

- **子主题**：
    - 中国实体 SIM 卡购买（中国移动/联通/电信外宾卡）
    - **eSIM 推荐对比**（Airalo / Nomad / Holafly / China Unicom eSIM）
    - 短期租 WiFi（漫游超人 / 环球漫游）
    - 本地运营商漫游套餐对比
- **变现**：Airalo/Nomad/Holafly 联盟（$1.50-2.50/单）⭐ 主要收入
- **窗口期警告**：2025 年 10 月 eSIM 才开始商业试点，窗口期 1-2 年

### 支柱 3：支付 ⭐（占 15%，2025-2026 痛点）

- **子主题**：
    - **支付宝外卡绑定教程**（Visa/Master/银联）
    - 微信支付外卡教程
    - 现金必要性（2026 现状：部分小店仍只收现金）
    - 银联卡 vs Visa/Master 接受度
    - 数字人民币（DCEP）外国人能否用
    - 港澳台/外籍华人特殊支付
- **变现**：数字支付联盟 + 银行开卡联盟
- **定位升级**：从「不可用」变为「能用但摩擦多」——用户需要「我的具体错误怎么解决」的排障指南

### 支柱 4：交通（占 12%）

- **子主题**：
    - 12306 高铁订票全流程（外国手机号注册、护照买票、改签）
    - 地铁通（每个城市单独一文）
    - 滴滴出行（DiDi）外国人注册
    - 共享单车（美团/哈啰/青桔）
    - 跨城交通（飞机 vs 高铁 vs 大巴）
    - 出租车防坑指南
- **变现**：Trip.com / 12306 海外版 联盟

### 支柱 5：必备 App（占 8%）

- **子主题**：
    - 高德/百度地图（vs Google Maps 不可用）
    - 翻译 App（DeepL / Google Translate / 微软翻译）
    - 订餐 / 外卖（美团/大众点评/饿了么）
    - 酒店（携程/飞猪/Booking/Agoda）
    - 出行（DiDi/12306/航旅纵横）
    - VPN 与「翻墙」的合规警告
- **变现**：App 推广联盟（CPI $1-5）

### 支柱 6：住宿（占 6%）

- **子主题**：
    - 携程 vs Booking vs Agoda vs Airbnb 对比
    - 外宾接待酒店清单（重要！中国限外宾入住特定酒店）
    - 青旅 / 民宿 / 短租
    - 酒店登记（外国护照登记流程）
- **变现**：Booking/Agoda/携程联盟（25-40% 佣金，实得 3.75-6%）

### 支柱 7：文化与禁忌（占 8%）

- **子主题**：
    - 餐厅礼仪（敬酒、转盘、AA 付账）
    - 宗教场所（清真寺/佛寺/教堂）礼仪
    - 不能谈的话题（政治、台藏疆）
    - 常见坑骗术（黑车/假门票/天安门周边骗局）
    - 数字礼仪（微信加好友、朋友圈点赞）
    - 排队 vs 插队文化
- **变现**：纯流量型，靠 AdSense

### 支柱 8：城市深度（占 12%，长尾 SEO 主力）

- **首批 10 城**：北京、上海、广州、深圳、成都、西安、杭州、重庆、桂林、丽江
- **每城标准结构**：
    - 城市概览 + 何时去
    - 机场到市区
    - 地铁/交通
    - 3-5 个必去景点
    - 当地美食
    - 1-2 天 / 3-5 天 / 7 天行程
    - 当地支付/通讯实操
    - 周边一日游
- **变现**：Trip.com 联盟 + 当地旅游局赞助

### 支柱 9：短期生存语言（占 4%）

- **子主题**：
    - 50 句关键中文（带拼音 + 音频）
    - 餐厅点餐常用语
    - 问路/打车/购物短语
    - 紧急情况（医院/警察/丢失护照）
    - 数字/价格/日期/时间表达
- **变现**：数字产品（PDF 短语手册 + 音频包 $4.9-9.9）

### 支柱 10：工具页（占 SEO 引流 + 留存 5%）

- 详见 [工具与社区](tools.md)
- **变现**：工具内嵌联盟链接

## 单篇文章 SOP

每篇正式内容必须包含：

1. **TL;DR 框**（3 行总结，开头）
2. **适用人群说明**（哪个国家、什么签证、停留多久）
3. **详细步骤/讲解**（带原始截图，每步 1-2 张图）
4. **费用清单**（表格，标注更新时间）
5. **FAQ 板块**（5-8 个长尾问题）
6. **相关工具内链**（3-5 个）
7. **相关文章内链**（3-5 个）
8. **作者署名 + 最后更新时间**

### 字数标准

| 页面类型 | 字数 |
|---------|------|
| 主题页 | 2500-3500 字 |
| 子主题页 | 1500-2500 字 |
| 城市页 | 1500-2000 字 |
| 工具页 | 800-1500 字 + 工具本身 |

### 图片标准

- 至少 5 张原创截图/照片（**绝不用 stock 图**）
- 每张图加 alt 标签（SEO + 无障碍）
- 关键步骤用红框标注
