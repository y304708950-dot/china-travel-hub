# 内容与 SEO

## 内容生产策略

### 生产流程

```
选题 → 大纲 → AI 初稿 → 人工改写/补充 → 截图/配图 → 审校 → 发布 → 内链优化
```

### 选题方法

1. **关键词研究**：Google Keyword Planner、Ahrefs、SEMrush
2. **竞品分析**：看竞品写了什么，找他们没覆盖的角度
3. **社区挖掘**：从 Pantip、Facebook 群、TikTok 评论区找真实问题
4. **政策追踪**：关注政策变化，快速产出时效性内容

### 内容生产效率（副业模式）

| 内容类型 | 生产时间 | 周产量 |
|---------|---------|--------|
| 深度指南（英文） | 3-4 小时 | 1-2 篇 |
| 排障教程（英文） | 1-2 小时 | 2-3 篇 |
| 泰语翻译 + 审校 | 0.5-1 小时 | 2-3 篇 |
| 工具页开发 | 4-8 小时 | 每月 1-2 个 |

**每周投入时间**：10-15 小时（副业可承受）

!!! tip "AI 提效，但不依赖 AI"
    - AI 是工具，不是替代品
    - 内容必须有人工改写和审校
    - 真实感和信任度是核心竞争力

## SEO 策略

### 技术 SEO 基础

- 静态站点生成（Astro），加载速度快
- 响应式设计，移动端友好
- 正确的 hreflang 标签（多语言）
- 结构化数据（Schema.org：Article / BreadcrumbList / FAQPage / HowTo）
- XML sitemap + robots.txt
- Google Search Console 提交

### 性能与 SEO 基线

- **Lighthouse 分数**：Performance 95+, SEO 100, Accessibility 95+
- **Core Web Vitals**：LCP < 2.5s, FID < 100ms, CLS < 0.1
- **首屏 < 1.5s**（全球访问）

### 关键词策略

=== "主关键词"
    竞争度中等，搜索量中等（"Alipay for foreigners"）

=== "长尾关键词"
    竞争度低，搜索量小但精准（"Krungthai card Alipay not working"）

=== "问题型关键词"
    "how to"、"what to do if"、"why" 开头的问题

=== "国家限定"
    "Thai tourist China visa"、"Vietnam eSIM China"

### 外链策略

- **联盟伙伴反向链接**：eSIM 供应商、OTA 平台的合作伙伴页面
- **Reddit/Quora 回答**：真实回答问题，适当带链接
- **客座文章**：给其他旅游博客写客座文章，带反向链接
- **社区签名**：论坛、群组签名档放链接

## 国际化与翻译流

### 翻译流（推荐）

1. 英文版原创
2. 翻译到第二语言（母语翻译者，不要用机翻）
3. 母语翻译者本地化（加当地场景）
4. 当地 beta reader 审校
5. 发布

### 翻译预算

- 母语翻译者：$0.05-0.15/词
- 一篇 2000 字文章翻译：$100-300
- 30 篇 MVP 内容翻译：$3,000-9,000（这就是为什么 1 个语言起步）

### 翻译工具链

- Crowdin 管理翻译记忆库
- 后期重复内容翻译成本可降 50-70%
- 关键术语用 glossary 统一（如「支付宝」始终翻译一致）

!!! warning "朋友做审校的可持续性"
    原方案朋友 2-3 小时/周、每篇 $20-40 辛苦费，**能否持续 6-12 个月是问号**。如果朋友退出，泰语内容质量立刻崩塌。
    
    **对策**：明确「朋友档期」和「备选翻译者」。Upwork 上找 1-2 个备用泰语译者，即使不用也先建立联系。
