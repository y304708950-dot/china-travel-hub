# 实施路线图

## 总体节奏

```
Phase 0（1-2周）→ Phase 1（3-12周）→ Phase 2（4-6月）→ Phase 3（7-12月）→ Phase 4（13-24月）
  准备期           MVP验证期          内容深化期          增长扩展期          规模化期
```

---

## Phase 0：准备期（第 1-2 周）

**目标**：完成基础搭建，验证市场需求

| 任务 | 说明 | 时间 | 成本 |
|------|------|------|------|
| 域名注册 | 选一个好记的 .com 域名 | 1 天 | $10-15 |
| 技术栈搭建 | Astro + Cloudflare Pages 部署 | 2-3 天 | $0 |
| 品牌基础 | Logo、配色、基础页面（关于、联系） | 1 天 | $0-50 |
| 关键词研究 | 50-100 个核心关键词地图 | 2 天 | $0 |
| 联盟申请 | Holafly、Trip.com、Agoda 等 | 1 天 | $0 |
| 市场验证 | 去 Pantip/TikTok 挖需求，发 3 个视频，建群看反馈 | 2 天 | $0 |

**G0 决策点（第 2 周）**：

- ✅ 技术栈正常运行
- ✅ 至少 1 个联盟审核中
- ✅ 视频有基础播放、群有人加入
- **预算**：$50-100

---

## Phase 1：MVP 验证期（第 3-12 周）

**目标**：发布核心内容和工具，验证核心假设

!!! warning "战略修正：只做 1 个语言"
    原方案「英语+泰语同时做」违反「集中力量打 1 国」原则。**Phase 1 只做泰语做透**，英语只作翻译源不主推。

| 任务 | 说明 | 工作量 |
|------|------|--------|
| 核心内容（泰语） | 支付 3 篇 + eSIM 2 篇 + 签证 2 篇 + 交通 2 篇 = 9 篇 | 30-40 小时 |
| 工具页开发 | eSIM 对比器 + 免签查询器 + 汇率换算器 = 3 个 | 20-30 小时 |
| SEO 基础配置 | 结构化数据、sitemap、GSC 提交 | 5-10 小时 |
| 联盟接入与追踪 | 联盟链接部署、转化追踪配置 | 5 小时 |
| 基础运营 | TikTok/Shorts 每周 2-3 个、社群维护 | 每周 3-5 小时 |

### 内容清单（首期 9 篇 + 3 工具）

| 类型 | 数量 | 选题示例 |
|------|------|----------|
| 支付排障 | 3 | 支付宝外卡绑定、微信支付、常见错误解决 |
| 通讯/eSIM | 2 | eSIM 对比指南、Airalo 购买教程 |
| 签证 | 2 | 泰国游客免签指南、签证类型 |
| 交通 | 2 | 12306 购票、打车攻略 |
| 工具页 | 3 | eSIM 对比器、免签查询、汇率换算 |

### G1 决策点（第 12 周）

!!! danger "门槛修正：原 UV≥200 太低"
    200 UV/月日均 6-7 个访问，可能全是自己测试 + 朋友访问，无法验证真实假设。**门槛提高到 UV≥500**。

| 指标 | 目标 | 权重 |
|------|------|------|
| 月有机 UV | ≥ 500 | 硬指标 |
| 联盟转化 | ≥ 5 单 | 硬指标 |
| Google 收录 | ≥ 10 篇 | 硬指标 |
| 自然搜索词进前 30 | ≥ 1 个（非朋友圈） | 硬指标 |
| 社群人数 | ≥ 50 人 | 软指标 |
| 视频播放 | 累计 ≥ 5000 | 软指标 |

- **Go**：4 项硬指标全部达标 → 进入 Phase 2
- **No-Go**：2 项以上不达标 → 调整方向或止损
- **最大损失**：$300-500
- **预算**：$300-500

---

## Phase 2：内容深化期（第 4-6 月）

**目标**：扩大内容体量，优化转化，建立用户反馈循环

| 任务 | 说明 |
|------|------|
| 内容扩展 | 泰语累计 30-40 篇 |
| 工具增加 | 支付错误码查询、紧急号码查询 |
| 转化优化 | A/B 测试 CTA、联盟链接位置、推荐文案 |
| 外链建设 | Reddit/Quora 回答、客座文章、联盟伙伴链接 |
| 邮件列表 | 订阅弹窗、定期 newsletter |
| 社群运营 | 群人数 200-300，每周有活跃讨论 |

### G2 决策点（第 24 周）

| 指标 | 目标 |
|------|------|
| 月有机 UV | ≥ 1,000 |
| 月联盟收入 | ≥ $50 |
| 邮件订阅 | ≥ 100 |
| 社群人数 | ≥ 200 |

- **Go**：核心指标达标 → 进入 Phase 3
- **No-Go**：差距大 → 延长 Phase 2 或调整策略
- **预算**：$500-1,000

---

## Phase 3：增长扩展期（第 7-12 月）

**目标**：泰语站成熟，启动越南语，多元化收入

| 任务 | 说明 |
|------|------|
| 泰语站深化 | 泰文累计 40-50 篇，建立泰语市场领先地位 |
| 越南语启动 | 翻译 15-20 篇核心内容，验证越南语市场 |
| 数字产品 | 推出 1-2 个 PDF 指南（城市离线包、支付排障手册） |
| 收入多元化 | 接入更多联盟（保险、景点门票、签证服务） |
| 品牌建设 | YouTube 频道、更多社媒渠道、建立专家形象 |
| 工具深化 | 12306 购票模拟器、菜单翻译器等高级工具 |

### G3 决策点（第 52 周）

| 指标 | 目标 |
|------|------|
| 月总 UV | ≥ 5,000 |
| 月总收入 | ≥ $300 |
| 越南语月 UV | ≥ 500 |
| 数字产品月收入 | ≥ $50 |

- **Go**：核心指标达标 → 进入 Phase 4
- **No-Go**：聚焦已有语言，暂缓扩展
- **预算**：$1,500-3,000

---

## Phase 4：规模化期（第 13-24 月）

**目标**：多语言矩阵，稳定盈利，探索退出路径

| 方向 | 说明 |
|------|------|
| **第三语言** | 印尼语/马来语，根据数据选择 |
| **内容深化** | 每语言 80-120 篇，覆盖更多长尾 |
| **工具矩阵** | 10+ 工具页，形成工具站品牌 |
| **收入多元化** | 联盟 + 数字产品 + AdSense + B2B |
| **社区规模化** | 多语言社群，总人数 3000+ |
| **退出路径探索** | 卖站、转型品牌、保持运营 |

### M24 目标

- 月 UV：15,000-30,000
- 月收入：$1,000-3,000
- 语言：2-3 种
- 社群：3000-5000 人

| 决策点 | 最大可承受损失 | No-Go 行动 |
|--------|-------------|-----------|
| G0（第 2 周） | $100 | 换方向或放弃 |
| G1（M3） | $500 | 重新评估语言/定位/产品方向 |
| G2（M6） | $1,500 | 延长深化期或调整变现模式 |
| G3（M12） | $4,000 | 聚焦已有语言，暂缓扩展 |
